# Checklist de despliegue

## Antes de subir
- [ ] npm install ejecutado sin errores
- [ ] npm run dev funciona
- [ ] npm run build funciona
- [ ] npm run preview funciona
- [ ] vite.config.js tiene el base correcto
- [ ] Las imagenes cargan bien
- [ ] Los estilos cargan bien
- [ ] Los CTAs funcionan
- [ ] No se modifico la app principal

## En Hostinger
- [ ] Entrar a hPanel
- [ ] Abrir File Manager
- [ ] Entrar a public_html
- [ ] Crear o abrir la carpeta correspondiente
- [ ] Subir el contenido de dist/
- [ ] Verificar que exista index.html
- [ ] Verificar que exista assets/

## Despues de subir
- [ ] Abrir la URL publica
- [ ] Verificar que carga la landing
- [ ] Verificar estilos
- [ ] Verificar imagenes
- [ ] Verificar CTAs
- [ ] Verificar en celular
